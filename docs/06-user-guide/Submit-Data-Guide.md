# Submit Data - User Guide

**Feature:** Data Collection (Phase 2)  
**Access:** `/data-points/submit`

---

## Quick Start

1. Allow GPS access when prompted
2. Select campaign and metric
3. Enter measurement value
4. (Optional) Add photo and notes
5. Click **Submit**

---

## GPS Capture

**Auto-capture:** Coordinates load automatically  
**Accuracy:** Displayed in meters (green = good, yellow = fair, red = poor)  
**Manual:** Use coordinate inputs if GPS fails

---

## Photo Upload

- Max 5MB
- Auto-geotag with coordinates
- Supported: JPG, PNG, GIF, WEBP

---

## Tips

- Wait for GPS lock before submitting
- Add notes for unusual readings
- Photos help validate outliers
- Check accuracy indicator before submit
