# Heatmap Analytics - User Guide

**Feature:** Heatmap Visualization (Phase 5)  
**Access:** `/analytics/heatmap`

---

## Quick Start

1. Select **Metric** (required)
2. (Optional) Filter by **Campaign**
3. View heatmap intensity gradient
4. Toggle **Map Type** (Street/Satellite)

---

## Reading the Heatmap

**Colors:**
- Blue = Low values
- Green = Medium values  
- Red = High values

**Intensity:** Brightness shows concentration  
**Auto-zoom:** Map fits to data bounds

---

## Statistics Panel

## Filters

**Metric:** REQUIRED - prevents mixing incompatible units  
**Campaign:** Optional - narrows to specific study area  
**Map Type:** Street view or Satellite imagery
