# Survey Map - User Guide

**Feature:** Geospatial Visualization (Phase 3)  
**Access:** `/maps/survey`

---

## Quick Start

1. Click **Campaign** dropdown to filter
2. Select **Metric** to visualize
3. Click markers to view details
4. Use zoom/pan to explore

---

## Map Features

**Markers:** Each data point shows location  
**Clustering:** Groups nearby points (click to expand)  
**Popups:** Click marker for reading details  
**Auto-zoom:** Fits all data points in view

---

## Filters

**Campaign:** Shows only selected campaign data  
**Metric:** Filters by measurement type (Temperature, Humidity, etc.)  
**Clear Filters:** Select "All" to reset

---

## Reading Details

Popup shows:
- Measurement value with unit
- Collection date/time
- GPS accuracy
- Campaign name
- Metric type
- Photo (if available)
