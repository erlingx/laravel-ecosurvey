# Campaign Management - User Guide

**Feature:** Admin Panel (Phase 1)  
**Access:** `/admin/campaigns`

---

## Create Campaign

1. Click **Create** button
2. Enter name and description
3. Set status (Draft/Active/Completed/Archived)
4. Click **Create**

**Owner:** Automatically assigned to you

---

## Manage Zones

1. Open campaign
2. Click **Actions → Manage Zones**
3. Draw polygons on map
4. Name and describe each zone
5. View calculated area

---

## Edit Campaign

1. Click **Edit** on campaign row
2. Update fields
3. Click **Save**

---

## Campaign Status

**Draft:** Planning stage  
**Active:** Collecting data  
**Completed:** Finished  
**Archived:** Historical reference

---

## Quick Actions

**Export Data:** Actions → Export (JSON/CSV)  
**Delete:** Edit page → Delete button (confirm required)  
**Search:** Use table search to find campaigns
