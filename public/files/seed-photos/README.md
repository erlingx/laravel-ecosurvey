# Seed Photos

This directory contains sample nature photos used by the database seeder for demonstration purposes.

## Files
- `forest-path.jpg` - Forest trail image
- `mountain-lake.jpg` - Mountain lake scene
- `tree-canopy.jpg` - Tree canopy view
- `misty-mountains.jpg` - Misty mountain landscape
- `green-meadow.jpg` - Green meadow scene

## Source
Photos sourced from Unsplash (free to use under Unsplash License).

## Usage
These photos are automatically assigned to seeded data points in:
- `database/seeders/EcoSurveySeeder.php`

The seeder randomly selects from these 5 photos when creating sample data.
